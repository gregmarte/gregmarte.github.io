A day in the life of a DevOps Engineering Manager (high level).

  • Own the AWS account and all resources within it.
  • Own the CI/CD pipelines.
  • Own the infrastructure as code (IaC) repositories.
  • Own the monitoring and alerting systems.
  • Own the security and compliance of the cloud environment.
  • Work closely with development teams to ensure smooth deployment processes.
  • Manage costs and optimize resource usage in the cloud.
  • Stay updated with the latest cloud technologies and best practices.
